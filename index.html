Fractal simulation

Change the number of container points (3-10) and observe changes in the fractal.
Next Point Fraction (2-5): The fraction of the connecting line between the current point and the
selected container point, by which the current point will move (e.g. 2=midpoint, 3=1/3 of the line etc.)
Click Randomize to view other variances of the simulation under the same parameters.
